w7 (2a 採 Youtube 影片簡報): 5-bar Linkage Plotter 與 Openduck mini 零組件設計繪製完成之後，如何切割零件? 為何要切割零件？ 如何利用 Webots 進行後續的控制系統設計與動態模擬 (Dynamic Simulation)？(plotter_project.7z)

為何要切割零件？如何切割？

  • 為何切割： 主要是為了 3D 列印 或 雷射切割。單一大型複雜模型難以直接列印，切割後可增加列印成功率並減少支撐材。

  • 如何切割： 在 CAD 中使用「Split」指令，或在 Slicer 軟體中依據列印床尺寸進行分割。

如何導入 Webots？

  1. 將 CAD 模型導出為 .stl 或 .obj。

  2. 在 Webots 中建立 Robot 節點，並在 Children 下新增 Shape 匯入模型。

  3. 設定物理特性（Mass, Inertia）以進行動態模擬。

w10: Webots 是什麼？有什麼特點？ 如何進行場景模擬？ 如何用 Python 進行控制系統設計？ (Tutorial)

  • 什麼是 Webots？ 一款開源的專業移動機器人模擬器，由 Cyberbotics 開發。
  • 特點：
    • 跨平台： Windows, Linux, macOS。
    • 多語言支援： Python, C++, MATLAB。
    • 感測器豐富： 內建 LiDAR, 攝影機, IMU 等。
  • Python 控制： 使用 controller 函式庫，透過 robot.getDevice() 取得馬達或感測器權限，並在 while 迴圈中進行時步（Time Step）更新。

w14: Webots 的模型有沒有辦法用程式方法建立? 有沒有辦法直接從 CAD 套件中轉出 Webots 的場景模型？

  • 透過 Webots 的 .proto 檔案格式。你可以撰寫類參數化的腳本，根據輸入參數自動生成不同的機器人尺寸。
  • CAD 轉出：
    • 可以使用專門的插件（如 URDF Exporter），將 Onshape 或 SolidWorks 模型轉為 URDF 格式，再由 Webots 匯入。
    • Webots 本身也支援直接匯入某些格式的 3D 模型作為節點。

w17: 如何實作一套 5-bar Linkage Plotter 與 Openduck mini？(Disney Bipedal Robot)

  • 5-bar Linkage Plotter：利用平行連桿機構，將兩個馬達的旋轉轉化為末端點的二維運動。
  • Openduck mini：小型化雙足結構，重點在於重心平衡與舵機（Servo）的同步控制。
  • 實作步驟： 3D 列印零件 $\rightarrow$ 組裝舵機 $\rightarrow$ 使用 ESP32/Arduino 控制 $\rightarrow$ 透過 Webots 測試步態。
